# 玲珑OS UI组件库

## 1. 概述

玲珑OS的UI组件库基于[shadcn/ui](https://ui.shadcn.com/)构建，并在此基础上进行封装和扩展，形成了一套符合玲珑OS设计语言、功能强大且易于使用的组件体系。本文档是为开发者提供的UI组件使用指南和参考。

## 2. 设计原则

- **一致性**: 所有组件在视觉风格、交互行为和API设计上保持高度一致。
- **可组合性**: 组件被设计为可以轻松组合，以构建更复杂的UI结构。
- **可访问性**: 遵循WAI-ARIA标准，确保所有组件对辅助技术友好。
- **主题化**: 支持通过CSS变量轻松定制主题，实现明暗模式切换和品牌色自定义。

## 3. 核心组件

### 3.1 窗口 (`Window`)

`Window`是玲珑OS桌面环境的核心交互单元，用于承载应用和插件的UI。

- **组件构成**:
  - `Window`: 根组件，管理窗口的位置、大小、层级和状态。
  - `WindowHeader`: 窗口头部，包含标题、图标和控制按钮（最小化、最大化、关闭）。
  - `WindowContent`: 窗口内容区域，用于渲染应用本身。
  - `ResizeHandles`: 窗口边缘的拖拽手柄，用于调整窗口大小。

- **使用示例**:
  ```tsx
  import { Window, WindowHeader, WindowContent } from '@/components/ui/window';

  function MyApp() {
    return (
      <Window title="My App" initialSize={{ width: 800, height: 600 }}>
        <WindowHeader />
        <WindowContent>
          <p>Hello, World!</p>
        </WindowContent>
      </Window>
    );
  }
  ```

### 3.2 按钮 (`Button`)

继承自`shadcn/ui`的`Button`，并增加了符合玲珑OS风格的预设样式。

- **变体**: `default`, `destructive`, `outline`, `secondary`, `ghost`, `link`.
- **使用示例**:
  ```tsx
  import { Button } from '@/components/ui/button';

  <Button>Click me</Button>
  <Button variant="destructive">Delete</Button>
  ```

### 3.3 输入框 (`Input`)

标准的文本输入框。

- **使用示例**:
  ```tsx
  import { Input } from '@/components/ui/input';

  <Input type="email" placeholder="Email" />
  ```

### 3.4 对话框 (`Dialog`)

用于显示模态对话框。

- **组件构成**: `Dialog`, `DialogTrigger`, `DialogContent`, `DialogHeader`, `DialogTitle`, `DialogDescription`, `DialogFooter`.
- **使用示例**:
  ```tsx
  <Dialog>
    <DialogTrigger asChild>
      <Button variant="outline">Edit Profile</Button>
    </DialogTrigger>
    <DialogContent>
      <DialogHeader>
        <DialogTitle>Edit profile</DialogTitle>
      </DialogHeader>
      {/* Form goes here */}
    </DialogContent>
  </Dialog>
  ```

## 4. 上下文菜单 (`ContextMenu`)

通过右键点击触发的菜单，用于提供与特定UI元素相关的操作。

- **组件构成**: `ContextMenu`, `ContextMenuTrigger`, `ContextMenuContent`, `ContextMenuItem`, `ContextMenuSeparator`.
- **使用示例**:
  ```tsx
  <ContextMenu>
    <ContextMenuTrigger className="flex h-[150px] w-[300px] items-center justify-center rounded-md border">
      Right-click here
    </ContextMenuTrigger>
    <ContextMenuContent>
      <ContextMenuItem>Copy</ContextMenuItem>
      <ContextMenuItem>Paste</ContextMenuItem>
    </ContextMenuContent>
  </ContextMenu>
  ```

## 5. 自定义与主题化

玲珑OS的UI库通过CSS变量实现主题化。你可以在全局CSS文件中覆盖这些变量来定制颜色、圆角、字体等。

- **核心变量示例** (`src/styles/globals.css`):
  ```css
  :root {
    --background: 0 0% 100%;
    --foreground: 222.2 84% 4.9%;
    --primary: 222.2 47.4% 11.2%;
    /* ... more variables ... */
  }

  .dark {
    --background: 222.2 84% 4.9%;
    --foreground: 210 40% 98%;
    --primary: 210 40% 98%;
    /* ... more variables ... */
  }
  ```
