## 📋 文档贡献指南

### 添加新文档

1. 在本目录下创建新的 Markdown 文件
2. 文件名使用 kebab-case 格式（如：`new-feature-guide.md`）
3. 在本 README.md 中添加文档链接

### 文档结构

推荐使用以下结构组织文档：

```markdown
# 文档标题

## 概述
简要说明文档内容

## 详细说明
详细内容...

## 示例代码
代码示例...

## 相关资源
- 链接1
- 链接2
```
