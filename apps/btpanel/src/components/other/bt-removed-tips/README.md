# 移除提示组件

## 功能说明
用于展示即将移除功能的警告提示

## 使用场景
- 即将下线的功能提示
- 需要引导用户联系客服的场景
- 重要变更通知

## 组件参数 (Props)
| 参数名       | 类型       | 默认值                     | 说明                     |
|--------------|------------|----------------------------|--------------------------|
| tips         | string     | '本功能将在下个正式版本被移除' | 自定义提示内容         |
| showSurvey   | boolean    | true                       | 是否显示联系客服入口   |

## 事件说明
| 事件名       | 说明                 | 回调参数       |
|--------------|----------------------|----------------|
| service-click | 点击联系客服时触发 | -              |

## 基本使用
```vue
<template>
  <bt-removed-tips 
    tips="当前功能将在v2.0版本移除"
    @service-click="handleService"
  />
</template>
```

## 样式覆盖
```scss
/* 修改警告图标颜色 */
.svgtofont-el-warning-filled {
  color: #e6a23c;
}

/* 调整提示文字大小 */
.bt-link {
  font-size: var(--el-font-size-base);
}
```

## 注意事项
1. 建议在功能下线前3个版本开始提示
2. 客服入口需配置在线客服系统
3. 重要提示建议保持不可关闭
4. 移动端需调整文字大小
5. 多语言场景需外部处理翻译
