# 二维码生成组件

## 功能说明
基于qrcode库封装的二维码生成组件，支持自定义样式和多种配置参数

## 使用场景
- 生成可扫描的二维码
- 需要自定义样式的二维码展示
- 动态更新二维码内容
- 不同容错率要求的场景

## 组件参数 (Props)
| 参数名        | 类型                  | 默认值       | 必填 | 说明                                                                 |
|---------------|-----------------------|--------------|------|--------------------------------------------------------------------|
| value         | string                | -            | 是   | 二维码内容（必填）                                                 |
| size          | number                | 100          | 否   | 二维码尺寸（像素）                                                 |
| margin        | number                | 0            | 否   | 二维码边距                                                         |
| level         | 'L'\|'M'\|'Q'\|'H'    | 'M'          | 否   | 容错等级：L(7%)/M(15%)/Q(25%)/H(30%)                               |
| scale         | number                | 4            | 否   | 缩放比例（每个模块的像素数）                                       |
| small         | boolean               | false        | 否   | 是否生成小尺寸二维码                                               |
| foreground    | string                | '#000000ff'  | 否   | 前景色（二维码颜色）                                               |
| background    | string                | '#ffffffff'  | 否   | 背景色                                                             |

## 功能特性
- 动态内容更新
- 响应式尺寸调整
- 多种容错等级选择
- 自定义颜色配置
- 高清二维码生成
- 自动渲染优化

## 使用示例
```vue
<template>
  <bt-qrcode 
    value="https://example.com"
    :size="200"
    foreground="#20a53a"
    background="#f5f7fa"
  />
</template>
```

## 样式覆盖
```scss
/* 调整二维码容器布局 */
.flex {
  justify-content: center;
  padding: 1rem;
}

/* 自定义二维码边框 */
canvas {
  border: 2px solid #e4e7ed;
  border-radius: 8px;
}
```

## 注意事项
1. 内容长度需符合二维码容量限制
2. 颜色值需使用有效CSS颜色格式
3. 小尺寸二维码可能影响扫描识别
4. 高清二维码建议使用较大size值
5. 动态更新内容时注意性能优化
6. 移动端需测试不同尺寸显示效果
7. 需要安装qrcode依赖库
