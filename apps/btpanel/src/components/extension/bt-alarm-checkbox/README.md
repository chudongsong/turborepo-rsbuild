# 报警复选框组件

## 功能说明
消息通道配置复选框组，支持通道状态检测和自动配置

## 使用场景
- 消息通知渠道配置
- 报警方式多选
- 通道状态检测
- 未配置通道快速安装

## 组件参数 (Props)
| 参数名     | 类型       | 默认值    | 必填 | 说明                          |
|------------|------------|-----------|------|-----------------------------|
| modelValue | string[]   | []        | 是   | 选中的通道列表               |
| name       | string     | 'give'    | 否   | 复选框组名称                 |
| limitType  | string[]   | ['sms']   | 否   | 需要排除的通道类型           |

## 事件说明
| 事件名   | 说明                 | 回调参数              |
|----------|----------------------|-----------------------|
| change   | 选中状态变化时触发   | 当前选中值数组        |

## 功能特性
- 自动检测通道配置状态
- 未配置通道快速安装
- 支持多种消息通道类型
- 状态提示与交互引导

## 使用示例
```vue
<template>
  <bt-alarm-checkbox 
    v-model="selectedChannels"
    :limit-type="['sms']"
  />
</template>
```

## 样式覆盖
```scss
/* 修改警告文字颜色 */
.text-danger {
  color: #f56c6c;
}

/* 调整复选框间距 */
.el-checkbox {
  margin-right: 1.5rem;
}
```

## 注意事项
1. 需要配合全局状态管理使用
2. 通道配置依赖后端接口
3. 微信公众号需要单独处理安装逻辑
4. 动态更新需调用refresh方法
