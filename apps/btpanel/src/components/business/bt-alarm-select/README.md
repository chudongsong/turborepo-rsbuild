# BtAlarmSelect 告警方式选择器

新版告警配置选择组件，支持动态过滤和配置状态显示。

## 功能特性
- 智能选项过滤
- 实时配置状态显示
- 自动同步选中值
- 配置跳转功能
- 多选模式支持

## 基本用法

```vue
<template>
  <bt-alarm-select 
    :multiple="true"
    :limit="['mail', 'webhook']"
    @change="handleSelectionChange"
  />
</template>
```

## 组件 API 说明

### Props
| 属性名       | 说明                | 类型      | 默认值   | 可选值                          |
|-------------|-------------------|-----------|---------|--------------------------------|
| multiple    | 是否多选模式         | boolean   | true    | true/false                    |
| limit       | 限制显示的告警类型   | string[]  | []      | mail/dingding/webhook 等       |

### 事件 (Events)
| 事件名     | 说明               | 参数类型       |
|-----------|------------------|---------------|
| change    | 选项变更时触发       | string[]      |

## 选项状态说明
| 状态样式          | 说明               |
|------------------|------------------|
| 正常选项          | 已配置且可用        |
| 禁用选项 (灰色)   | 未配置或不可用       |
| [未配置] 标签     | 红色文字提示        |
| [已配置] 标签     | 可点击跳转配置       |

## 核心功能
1. **智能过滤**：根据 limit 属性动态过滤选项
2. **值同步**：自动清理无效选中值
3. **配置跳转**：点击标签直接进入配置界面
4. **状态显示**：实时反映配置完成状态

## 配置类型对照表
| type 值       | 对应渠道       | 配置组件    |
|---------------|--------------|-----------|
| mail          | 邮件通知       | Mail      |
| dingding      | 钉钉机器人     | Common    |
| feishu        | 飞书机器人     | Common    |
| weixin        | 企业微信机器人  | Common    |
| webhook       | 自定义Webhook | WebHook   |
| sms           | 短信通知       | 特殊配置    |

## 最佳实践
1. **动态过滤**：根据业务场景动态设置 limit
2. **值校验**：组件自动处理无效选中值
3. **配置引导**：未配置项显示快速入口
4. **状态同步**：通过 @change 事件保持数据一致

## 关联接口
- `getSenderAlarmListInfo`：获取告警配置列表
- `setAlarmModuleDialog`：打开配置对话框
- `installAlarmModule`：安装依赖模块
