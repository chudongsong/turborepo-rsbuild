# BtBindUser 宝塔账号绑定组件

用于绑定/切换宝塔官网账号的表单组件，支持验证码验证和账号切换功能。

## 功能特性
- 账号绑定/切换功能
- 表单验证功能
- 验证码动态获取
- 自动聚焦输入框
- 错误状态提示
- 快捷链接导航

## 基本用法

```vue
<template>
  <bt-bind-user :comp-data="{ title: '绑定宝塔账号', btn: '立即绑定' }" />
</template>
```

## 组件 API 说明

### Props
| 属性名       | 说明                | 类型      | 默认值   | 可选值                          |
|-------------|-------------------|-----------|---------|--------------------------------|
| compData    | 配置数据            | object<{title?: string, btn?: string}> | {}     | title(标题文本)/btn(按钮文案) 的配置对象           |

### 事件 (Events)
| 事件名     | 说明               | 参数类型       |
|-----------|------------------|---------------|
| onOpen    | 弹窗打开时触发       | —             |
| onCancel  | 弹窗关闭时触发       | —             |

## 表单验证规则
| 字段名     | 验证规则                          | 错误提示                  |
|-----------|---------------------------------|-------------------------|
| username  | 手机号格式验证                     | 请输入正确的手机号          |
| password  | 长度≥8位                        | 请输入至少8位密码           |
| code      | 6位数字验证（需开启验证码）         | 请输入6位验证码            |

## 操作说明
1. **账号输入**：支持手机号格式验证
2. **密码输入**：至少8位字符
3. **验证码获取**：需先填写手机号和密码
4. **表单提交**：自动处理绑定/切换逻辑

## 状态管理
| 状态类型       | 触发条件               | 表现方式          |
|---------------|----------------------|------------------|
| 验证码倒计时   | 点击获取验证码          | 60秒倒计时显示      |
| 表单禁用状态   | 验证码获取期间          | 输入框禁用         |
| 错误提示状态   | 手机号格式错误          | 红色错误提示        |

## 最佳实践
1. **自动聚焦**：组件打开时自动聚焦用户名输入框
2. **表单重置**：关闭弹窗时自动清空验证状态
3. **安全验证**：验证码需先验证手机号和密码
4. **错误处理**：统一处理接口返回错误信息

## 关联接口
- `getBindCode`：获取短信验证码
- `bindUserRequest`：提交绑定请求
- `getGlobalInfo`：获取全局配置信息
