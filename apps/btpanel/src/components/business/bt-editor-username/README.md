# BtEditorUsername 用户名编辑器

用于修改系统用户名的安全验证表单组件。

## 功能特性
- 双重输入验证
- 即时格式校验
- 自动页面刷新
- 紧凑布局设计
- 防误操作机制

## 基本用法

```vue
<template>
  <bt-editor-username />
</template>
```

## 组件事件

| 事件名         | 说明                          | 参数                  |
|-------------|-----------------------------|-------------------------|
| resize   | 组件大小调整事件               | —                         |
| validateError | 验证失败事件               | 错误字段信息 object      |

## 验证规则
| 字段         | 规则                          | 错误提示                  |
|-------------|-----------------------------|-------------------------|
| username1   | 必填且≥4位                   | "用户名长度必须大于3位"    |
| username2   | 必须与 username1 一致         | "两次输入的用户名不一致"    |

## 安全机制
1. **提交验证**：表单验证通过后才发送请求
2. **自动退出**：修改成功后强制刷新页面
3. **防重复提交**：请求期间禁用按钮
4. **输入屏蔽**：禁止特殊字符输入
5. **审计日志**：记录用户名修改操作
6. **会话保护**：修改后清除敏感缓存

## 最佳实践
1. **使用场景**：在系统设置中调用
2. **权限控制**：仅管理员可访问
3. **输入引导**：提示用户名规范
4. **错误处理**：统一处理接口错误

## 关联接口
- 用户名修改：`/system?action=set_username`

> 需配合权限管理系统使用
