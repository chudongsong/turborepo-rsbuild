# BtErrorMask 错误遮罩组件

系统级错误提示组件，用于处理磁盘空间不足等严重错误。

## 功能特性
- 全局遮罩覆盖
- 错误代码解析
- 自动跳转引导
- 视觉强调设计
- 多错误类型支持

## 基本用法

```vue
<template>
  <bt-error-mask :options="{ code: 'database is read only!' }" />
</template>
```

## 配置参数

| 参数名         | 类型     | 默认值       | 说明                |
|-------------|---------|-------------|-------------------|
| options     | object  | {}          | 错误配置对象{code: string}       |
| autoRedirect| boolean | true        | 是否自动跳转清理页面     |

## 错误类型
| 错误代码                   | 说明                | 处理建议              |
|--------------------------|-------------------|---------------------|
| database is read only!    | 磁盘满导致数据库只读     | 清理系统根目录空间       |
| unable to open database file | 无法打开数据库文件      | 检查磁盘权限与空间       |

## 视觉设计
| 元素             | 样式特征                      |
|------------------|-----------------------------|
| 遮罩层           | 70% 透明度白色背景             |
| 错误框           | 橙色警告图标+阴影效果            |
| 操作按钮         | 品牌绿色主题色+悬浮效果           |

## 最佳实践
1. **错误代码映射**：维护 errorInfo 配置错误信息
2. **类型扩展**：通过 errorType 支持更多错误场景
3. **自动跳转**：磁盘错误时引导至文件管理
4. **全局使用**：在布局组件中动态挂载
5. **多语言支持**：根据系统语言切换提示文案
6. **日志记录**：自动上报错误信息到监控系统
