# BtCustomerService 客服组件

集成在线客服和用户反馈功能的侧边工具栏组件。

## 功能特性
- 悬浮式侧边工具栏
- 微信客服二维码展示
- 用户反馈入口
- 智能显隐控制
- 交互动画效果
- 响应式布局

## 基本用法

```vue
<template>
  <bt-customer-service />
</template>
```

## 组件结构
```text
├── 客服按钮
│   ├── 微信客服弹窗
│   │   ├── 客服二维码
│   │   ├── 关闭按钮
│   │   └── 文字说明
│   └── 用户反馈按钮
```

## 交互说明
1. **悬浮显示**：鼠标悬停显示客服弹窗
2. **点击固定**：点击客服图标固定弹窗
3. **自动隐藏**：鼠标移出后延迟关闭
4. **反馈入口**：点击反馈图标打开问卷

## 样式配置
| 类名                 | 说明                     | 示例值                     |
|---------------------|------------------------|--------------------------|
| .toolbar-right      | 侧边工具栏容器样式         | right:0; bottom:70px      |
| .wechat-customer    | 客服弹窗容器样式           | width:160px; height:214px |
| .customer-title     | 客服标题区域样式           | background:rgba(32,165,58,0.1) |
| .icon-menu          | 图标按钮基础样式           | width:35px; height:35px   |
| .icon-menu:hover    | 按钮悬停状态样式           | background:#20a53a       |

## 最佳实践
1. **位置固定**：组件始终固定在右侧中间位置
2. **颜色统一**：使用品牌绿色(#20a53a)作为主色调
3. **响应速度**：设置500ms延迟关闭提升体验
4. **外部链接**：客服链接指向官方客服页面
