# 修复方案组件

## 功能说明
提供系统修复的两种解决方案，支持手动修复指引和付费客服快速通道

## 使用场景
- 系统检测到安全风险后的修复建议
- 需要提供多套解决方案的场景
- 需要引导用户联系客服的场景
- 技术问题快速解决通道

## 组件参数 (Props)
本组件为静态展示组件，无需传入参数

## 功能特性
- 双方案对比展示（手动/付费）
- 客服二维码集成展示
- 响应式布局适配
- 推荐方案醒目提示

## 基本使用
```vue
<template>
  <bt-recovery-plan />
</template>
```

## 样式覆盖示例
```css
/* 修改推荐方案提示颜色 */
.plan .title span {
  color: #ef9f00;
}

/* 调整二维码边框样式 */
.qrcode {
  border-radius: 0.4rem;
  padding: 0.5rem;
}
```

## 注意事项
1. 二维码图片路径为 'customer/customer-qrcode.png'，需确保资源存在
2. 推荐方案中的"付费修复"文案需根据实际情况调整
3. 二维码区域尺寸固定为 10rem×10rem，需保证图片清晰度
4. 组件采用固定内边距（3.4rem），外层容器需预留足够空间
5. 当前为静态组件，如需动态内容需扩展 props
