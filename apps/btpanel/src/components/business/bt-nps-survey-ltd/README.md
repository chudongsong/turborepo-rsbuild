# BtNpsSurveyLtd 企业版用户调研组件

## 功能特性
- 🏢 企业版专属调研表单
- 📝 多问题动态渲染
- ✅ 实时表单验证
- 📤 调研数据提交
- 🎉 提交成功反馈

## 组件对比
| 特性               | bt-nps-survey-ltd     | bt-nps-survey         |
|--------------------|-----------------------|-----------------------|
| 使用场景           | 企业版用户反馈         | 普通用户NPS评分       |
| 数据收集           | 文本建议为主           | 评分+建议结合         |
| 问题类型           | 开放式问题             | 评分+选择题           |
| 提交参数           | product_type=19       | product_type=0        |
| 反馈机制           | 本地存储标记           | 弹窗感谢              |

## 数据结构
### 参数说明
| 参数名       | 类型      | 必填 | 默认值   | 说明              |
|-------------|-----------|------|---------|-------------------|
| id          | number    | 是   | -       | 唯一问题标识      |
| question    | string    | 是   | -       | 问题正文内容      |
| hint        | string    | 否   | -       | 输入框提示文字    |
| required    | boolean   | 否   | false   | 是否必填项        |

## 交互事件
| 事件名       | 说明                 | 回调参数       |
|-------------|----------------------|---------------|
| close       | 关闭调研窗口时触发    | 无            |

## 核心方法
| 方法名        | 说明                 |
|--------------|----------------------|
| submit       | 表单数据提交         |
| validate     | 表单验证             |

  // 关闭后刷新列表等操作

## 接口说明
### 提交参数
```typescript
{
  questions: string   // 问题答案(JSON字符串)
  product_type: 19    // 固定为企业版标识
  software_name: 'Enterprise-Edition-Survey' // 版本标识
  rate: 0             // 固定为0(无评分)
}
```

## 样式定制
| CSS类名              | 说明                  |
|----------------------|----------------------|
| nps_survey_banner    | 企业版专属横幅样式     |
| el-form-item        | 问题项容器样式        |

## 注意事项
1. 依赖`getNpsQuestion`接口获取问题列表
2. 提交成功后会设置本地存储标记(ltd_nps=1)
3. 仅显示企业版专属调研问题
4. 第一个问题为必填项
5. 输入内容限制500字符
