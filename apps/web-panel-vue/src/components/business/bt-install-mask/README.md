# BtInstallMask 安装遮罩组件

用于提示未安装必要插件的全局遮罩组件，支持多插件状态显示。

## 功能特性
- 全局遮罩覆盖
- 多插件状态显示
- 一键安装引导
- 自定义内容插槽
- 自动定位居中

## 基本用法

```vue
<template>
  <bt-install-mask :options="{ name: 'nginx', title: 'Nginx服务' }" />
</template>
```

## 配置参数
| 参数名       | 类型               | 默认值   | 说明                          |
|-------------|-------------------|---------|-----------------------------|
| options     | object \| array   | {}      | 插件信息对象或数组 {name, title} |
| icon        | boolean          | true    | 是否显示警告图标                |
| width       | string           | auto    | 遮罩内容宽度                   |
| height      | string           | auto    | 遮罩内容高度                   |

## 插槽配置
| 插槽名       | 说明                          |
|-------------|-----------------------------|
| content     | 自定义安装提示内容            |


## 核心交互
1. **多插件显示**：数组形式显示多个未安装插件
2. **安装跳转**：点击"立即安装"打开插件管理
3. **内容定制**：通过插槽自定义显示内容
4. **自动居中**：始终居中显示在可视区域

## 最佳实践
1. **动态检测**：在功能模块挂载时自动检测依赖
2. **批量处理**：同时提示多个缺失插件
3. **样式覆盖**：使用插槽定制企业版样式
4. **状态同步**：安装后自动刷新页面状态

## 关联接口
- 插件信息获取：`/plugin?action=get_info`
- 插件安装接口：`/plugin?action=install`

> 需配合插件管理系统使用
